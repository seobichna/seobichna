NCS PROCESS

6개월 동안 NCS 과정을 수료하여 사용자 특성을 고려한 UI/UX 설계와 기획에 대한 폭넓은 이해를 얻었습니다.
스크롤을 내리면 상세 내용을 확인할 수 있습니다.

  • 01.Web UI UX Planning & Design

    STEP 01

    프로젝트 배경에 대해 조사하고 해당 서비스를 이용하는
    사용자 분석을 통해 콘셉을 결정 한 후 그 결과에 맞춰 주요 기능과 콘텐츠를 정의하여 계획을 수립 할 수 있습니다.

  • 02.Web UI implemented & Test eb UI planning

    STEP 02

    웹 UI를 효율적으로 테스트하기 위한 계획을 수립하고 테스트 환경을 설정하여 테스트 케이스와 시나리오를 작성하여 그 목적에 맞는 디자인과 기능을 구현합니다.

  • 03.Web UI Design

    STEP 03

    사용자 인터페이스를 설계하는 과정으로 사용자들이 웹
    페이지를 효과적으로 이용하고 상호 작용할 수 있도록
    디자인하는 것을 목표로 합니다.

  • 04.Web Frontend Development Basic

    STEP 04

    웹페이지의 구조와 콘텐츠를 정의하는 마크업 언어와
    스타일을 디자인하는 css 그리고 JavaScrip를 활용하여
    웹 동작과 성능 향상을 위한 기초 개념을 알게됩니다.

  • 05.Frontend Development React JS

    STEP 05

    ReactJS는 UI를 구축하는 컴포넌트 기반의 JavaScript
    라이브러리로 간결한 문법으로 동적인 웹 애플리케이션을 가장 효과적으로 개발 할 수 있도록 활용할 수 있습니다.

  • 06.Node Js Distributione

    STEP 06

    Node.js의 Distribution 은 runtime을 설치하고 설정하는 방법으로 공식 다운로드나 패키지 매니저/NVM/Docker를 사용하여 Node.js를 설치하고 관리합니다.

  • 07.Graphic design figma / phtoshop

    STEP 07

    Figma는 협업과 프로토타이핑에 Photoshop은 이미지 편집과 디자인에 사용되며 프로젝트에 맞게 선택하여 함께 활용하면 더 다양한 디자인 작업이 가능합니다.
